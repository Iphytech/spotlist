# Spotlist API

> A platform in which people can generate and share playlists for their favourite artists providing them with visibility

## Usage

Update .env file with the values/settings of your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run test
npm run test
```
